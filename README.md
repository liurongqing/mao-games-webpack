## 复制一份新项目
```bash
# 复制一份新项目，排除 .git 目录
rsync -av --exclude .git mao-game-webpack/ new-game
```

## 更新 npm 包

```bash
yarn upgrade-interactive --lastest # 空格: 选择  i: 反选  a: 全选 会更新 yarn.lock 文件，不会更新package.json文件
yarn upgrade phaser@lastest # 更新到最新，并更新 package.json 文件
```
